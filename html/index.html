STORAGE AND COMPUTE.

3Nodes form a global of decentralized storage and compute. Connections between 3Nodes are fully encrypted and all workloads are managed by a Smart Contract for IT. This combined provides similar security and reliability as a Blockchain, but at the performance of high-end server technology.
