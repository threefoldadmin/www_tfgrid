Ten thousand more 3nodes in 2020.

The ThreeFold Grid is currently distributed across 21 countries, has more thar 80 Million GB of storage and over twenty thousand CPU cores live. Become a ThreeFold Farmer and join us in the journey towards true decentralization.

Become a farmer

Become a part of the new Internet and host a node within the ThreeFold Grid. All
you need is an internet connection and a power socket. Decentralizing the
internet by becoming a farmer can be done in multiple ways.

diy farming

Buy or build any 3Node that can run on Zero OD (can be
new or refurbished hardware) in the any quantity. DIY
allows flexibility for connecting too the ThreeFold Grid.

Certified capacity

ThreeFold's certified hardware is completely locked down
and secure. They run capacity, network and compute at a
higher cloud production rate than DIY 3Nodes.

characteristics of the largest decentralized internet grid

The ThreeFold Grid already is the largest decentralized internet grid at this moment. The grid can run any Linux
workload but with more speed, cost-efficiency, privacy & security, sustainability. Soon the Grid will be open to
all individuals and companies to be used to run production workloads.

decentralized

Consists of 3Nodes - affordable, secure and highly efficient cloud storage units that can be purchased and hosted by anyone.

efficient

Algorythms ensure that all data is hosted where it is needed in order to reduce redunduncy.

resilient

The self-healing operating system ensures that in the unlikely case that a node goes down, your data remains safe and available.

fair

Farmers are remunerated and incentivized to constantly expand the ThreeFold Grid by generating ThreeFold Tokens (TFT) for the capacity provided.

secure

A stirage algorithm inspired by space-technology is used and ensures the integrity of all data, even if some nodes go offline.

sustainable

Being decentralized and using the greenest hardware available, the ThreeFold Grid is able to save 90% energy compared to conventional data centers.

Affordable

Data hosted on the ThreeFold Grid costs 70% less compared to other cloud providers.

tamper-proof

algorithms prove that workdloads on the ThreeFold Grid are executed as intended.

Autonomous

Deploys and operates the infrastructure without human interaction.
